# G2-Final-Project

This final project of creating a Banking App with Two main Programing Language:
  1. Java for Backend
  2. Javascript for Frontend
      - React Js
      - React Native

Main Feature on App:
  1. Registration
  2. Login
  3. Logout
  4. View Balance
  5. Telephone/ Telkom
      - Check Bill
      - Payment
  6. Postpaid Phone
      - Check Bill
      - Payment
